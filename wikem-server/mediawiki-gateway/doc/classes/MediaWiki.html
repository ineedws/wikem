uri_to_wiki(uri)

Convert URL-ized page name ("getting_there_%26_away") into Wiki display format page name ("getting there & away"). Also strips out any illegal characters (#<>[]|{}, cf. meta.wikimedia.org/wiki/Help:Page_name#Restrictions).
